# Mina zkApp: Price Feed Oracle

* Deploy `Oracle` and `DeFi`
Run interact.ts.
Then set the oracle publicKey in interactDefi.ts and run it.

This template uses TypeScript.

## How to build

```sh
npm run build
```

## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

```sh
npm run coverage
```
